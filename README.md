Aplicação CRUD de Futebol. Esta é uma aplicação simples em PHP para gerenciamento de times, jogadores e partidas de futebol, usando XAMPP. Requisitos: XAMPP instalado com Apache e MySQL, navegador web. Instalação: 1. Instalar XAMPP do site oficial. 2. Iniciar Apache e MySQL no painel do XAMPP. 3. No phpMyAdmin, criar banco 'futebol_db' e executar o script db/db.sql. 4. Copiar projeto para htdocs, verificar includes/db.php. 5. Acessar via http://localhost/pasta/index.php. Funcionalidades: Times - criar, listar com filtro nome e paginação, editar, excluir com bloqueio dependências. Jogadores - criar com validação posição e camisa, listar com filtros, editar, excluir. Partidas - criar impedindo mesmo time, listar com filtros, editar, excluir. Teste: Navegue, adicione, edite, filtre, exclua. Verifique erros. Notas: Se erro conexão, verifique MySQL e banco. Simples, procedural, sem frameworks.
